# Anthropic Messages

The planned Messages pack keeps version headers, content blocks, tool-use and
tool-result lifecycle, streaming event order, stop reasons, usage, and unknown
events distinct. OpenAI field names or tool semantics are not projected onto
Anthropic behavior as normative requirements.

Python and Node client behavior will be tested through version-pinned drivers
and profiles. No stable Anthropic pack or client support claim is published
yet.
