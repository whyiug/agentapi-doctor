# Naming

The project uses **AgentAPI Doctor** as its public working name and
`agentapi-doctor` for its repository, CLI, module, archives, and image paths.

The generic name `agentapi` is not used as a standalone CLI or package identity
because other projects already use it. Automated namespace or domain checks do
not constitute trademark clearance. Any future rename or trademark policy
change follows the normal RFC and maintainer review process.
