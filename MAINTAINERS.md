# Maintainers

This file records project roles that have actually been assigned.

| GitHub identity | Role | Scope |
|---|---|---|
| [@whyiug](https://github.com/whyiug) | Lead maintainer | Repository administration, review, releases, and private security intake |

CODEOWNERS routes review requests to the current maintainer. It does not imply
external review or endorsement.

Contributors interested in maintaining a protocol pack, driver, core area,
documentation, community support, or security response should begin with
sustained contributions in that area. Appointment and access changes follow
[GOVERNANCE.md](GOVERNANCE.md) and are recorded here when they occur.
