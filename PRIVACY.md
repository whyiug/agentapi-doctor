# Hosted Service Privacy Notice — Draft, Not in Effect

> **DRAFT ONLY.** There is no AgentAPI Doctor hosted Registry, public runner,
> telemetry collector, or managed service today. This notice has not been
> legally reviewed and does not describe a live data controller or processor.

## Current repository and local tool

Local use is governed by [DATA_POLICY.md](DATA_POLICY.md). The project does not
currently receive local run data through its own service. GitHub independently
processes repository activity under GitHub's own terms and privacy statement;
the project does not restate or extend those policies here.

## Requirements before hosted launch

Before any hosted processing begins, an effective privacy notice must disclose:

- the operator and real privacy contact;
- data controller/processor roles;
- categories collected, including account, signature, audit, artifact,
  observation, network, and support data;
- purposes and applicable legal bases;
- sources of data and whether fields are optional;
- recipients, subprocessors, storage regions, and cross-border transfers;
- retention and backup-deletion schedules by data class;
- access, correction, export, objection, withdrawal, dispute, and deletion
  procedures;
- automated decision-making or ranking, if any;
- security and incident-notification practices;
- age or jurisdiction restrictions; and
- notice, versioning, and consent requirements for material changes.

The public fact projection must remain separate from identities, signatures,
comments, raw artifacts, model text, and private configuration. Withdrawal and
physical deletion must account honestly for immutable audit facts and backup
expiry without claiming impossible instantaneous erasure.

## No current privacy request channel

Because no project-operated hosted service or user database exists, there is no
hosted-service privacy request process today. Sensitive repository material
should be reported privately under [SECURITY.md](SECURITY.md). A real privacy
contact and request workflow must be published before hosted launch.
